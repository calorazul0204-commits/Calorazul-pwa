PWA CALORAZUL - COMO VER NO COMPUTADOR

1. Descarregue e extraia o ficheiro ZIP.
2. Abra a pasta calorazul-pwa.
3. Clique duas vezes no ficheiro index.html.

IMPORTANTE:
Para instalar no telemóvel como app, a PWA precisa de estar online num alojamento com HTTPS.
Pode usar opções gratuitas como Netlify, Vercel ou GitHub Pages.
